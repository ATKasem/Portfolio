# Portfolio

Personal portfolio and blog for Aaron Thomas, a software engineer building AI tools, Go services, and thoughtful web experiences.

It's a static site built with plain HTML, CSS, and JavaScript. No framework, no build step. The `blog/` section holds write ups documenting the projects I build.

## Structure

```
index.html      Landing page (hero, projects, about, contact)
styles.css      Site styles
main.js         Interactions
blog/
  index.html    Blog index
  *.html        Individual posts
```

## Running locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
